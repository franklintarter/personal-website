# Franklin Tarter's Personal Website

## Required Tools

- [Node](https://nodejs.org/en/) & [NVM](https://github.com/nvm-sh/nvm)
  - `curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.0/install.sh | bash`
  - `nvm install 12.13.1`
  - `nvm use 12.13.1`

## Setup

```shell
yarn
yarn develop
```
